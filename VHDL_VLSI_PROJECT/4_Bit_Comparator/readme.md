# 4-bit Comparator Design
## Description
This project features the RTL implementation of a 4-bit Magnitude Comparator using VHDL. Developed with a Behavioral modeling style, the design provides an efficient solution for digital magnitude comparison, where two 4-bit binary numbers are evaluated to determine their relative values. The codebase is optimized for synthesis and includes a verification stimulus in the testbench to test cases for equality, as well as greater-than and less-than conditions.

## Theory and Functional Logic
A magnitude comparator is a combinational circuit that compares two digital words, A and B, to produce a logical output based on their relationship.
