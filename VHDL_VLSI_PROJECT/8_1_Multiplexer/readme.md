# 8:1 Multiplexer 
## Description
This project focuses on the design and implementation of an 8-to-1 Multiplexer optimized for the Xilinx Vivado environment. Developed with a Behavioral modeling style, the design manages eight input data lines using a 3-bit selection vector. This implementation highlights efficient resource utilization during RTL synthesis and includes a comprehensive testbench to ensure functional correctness across all input-output paths. 
# Theory and Functional Logic
An 8:1 MUX expands the data selection capability by utilizing 3 selection lines to manage 8 inputs ($2^3 = 8$). It is commonly used in telecommunications and computer memory systems to increase the amount of data that can be sent over a network within a specific timeframe.
