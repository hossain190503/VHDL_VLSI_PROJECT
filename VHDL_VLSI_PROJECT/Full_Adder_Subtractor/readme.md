# Full-Adder and Full-Subtractor

## Description
This project features the RTL implementation of a Full-Adder and Full-Subtractor using VHDL. Developed with a Behavioral modeling style, these circuits are designed to handle 3-bit inputs (including carry-in and borrow-in), providing a complete solution for multi-bit binary arithmetic. The codebase is optimized for synthesis and includes a testbench to verify the logic for all 8 possible input combinations.

## Theory and Functional Logic
Unlike half-arithmetic units, a Full-Adder incorporates a Carry-in (Cin) from a previous stage to produce a Sum and Carry-out. Similarly, a Full-Subtractor accounts for a Borrow-in (Bin) to calculate the Difference and Borrow-out. These circuits are the building blocks of Ripple Carry Adders and more complex ALU structures.
