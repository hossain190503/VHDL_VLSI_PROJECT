# XOR Gate Design
## Description
This project features the RTL implementation of an Exclusive-OR (XOR) gate using VHDL. Developed with a Behavioral modeling style, the design focuses on the unique logic where the output is high only when the inputs are different. This gate is a critical component for parity checkers, error detection circuits, and arithmetic adders.

## Theory and Functional Logic
An XOR gate is a digital logic gate that gives a true output ('1') whenever the number of true inputs is odd. For a 2-input XOR gate, the output is high if one, and only one, of the inputs is high. If both inputs are the same (both '0' or both '1'), the output is low.
