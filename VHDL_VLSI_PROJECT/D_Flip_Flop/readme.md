# D Flip-Flop
## Description
This project features the RTL implementation of a D Flip-Flop with an asynchronous reset. Developed with a Behavioral modeling style, this design represents the fundamental storage element in digital electronics. It is capable of latching a single bit of data on the rising edge of the clock, providing a reliable method for data synchronization and storage in sequential circuits.

## Theory and Functional Logic
The D (Data or Delay) Flip-Flop captures the value of the input at a precise moment (the clock edge) and holds that value until the next cycle. This behavior is essential for building registers and memory units. The inclusion of an asynchronous reset ensures the output can be forced to a known state (0) regardless of the clock.
